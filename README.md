# gophkeeper

Приложение _*gophkeeper*_ предназначено для хранения приватных данных пользователя.

Состоит из клиентской и серверной части.

*Клиентская часть* - обеспечивает:

- регистрацию нового пользователя;

- авторизацию зарегистрированного пользователя;

- загрузку данных, авторизованным пользователем;

- просмотр данных, авторизованным пользователем;

- удаление данных, авторизованным пользователем.

*Серверная часть* обеспечивает:

- хранение информации о пользователях;

- хранение приватных данных пользователей.

Шифрование данных осуществляется на стороне клиента, перед отправкой данных на сервер.

*Ответсвенность* за сохранность пароля шифрования лежит *на пользователе*.

Ограничение на размер сохраняемых файлов - 50 Мб.

Взаимодействие клиента и сервера осуществляется посредством сообщений. Сообщение содержит в себе зашифрованные данные (слайс байт) и метаинформацию, описывающую данные.

Соединение между клиентом и сервером устанавливается с использованием TLS.

Клиент собирается при помощи команды make. Может быть собран под ОС Linux, Windows, MacOS.
